## 概述

GitHub 是全球最大的代码托管平台和开发者社区，2008 年创立，2018 年被 Microsoft 收购。GitHub 深度投入 AI 编程辅助领域，推出了 GitHub Copilot 系列，包括 IDE 插件和终端助手 Copilot CLI。Copilot CLI 基于 [[Agentic Loop]] 模式运作，支持 [[MCP协议]] 扩展，是 [[AI编程助手]] 领域的重要参与者。

## 关键信息

### 核心产品

| 产品 | 类型 | 描述 |
|------|------|------|
| GitHub Platform | 代码托管 | Git 仓库托管、CI/CD（Actions）、项目管理 |
| GitHub Copilot | [[AI编程助手]] | IDE 内 AI 编程助手，支持多种编辑器 |
| GitHub Copilot CLI | [[AI编程助手]] | 终端 AI 助手，具备工程代理能力，支持 MCP 扩展 |
| GitHub Models | AI 模型 | 提供 AI 模型 API 访问 |

### Copilot CLI 的核心能力

- **工程代理**：自主执行多步骤工程任务，如代码重构、Bug 修复、功能开发
- **[[MCP协议]] 支持**：连接自定义 MCP 服务器扩展能力
- **GitHub 生态集成**：与 Issues、Pull Requests、Actions、Codespaces 深度联动
- **终端原生**：直接在命令行中辅助开发操作
- **Skills 和 Hooks**：支持可复用技能和自定义钩子

### 竞争格局

- 与 [[Anthropic]] 的 Claude Code 构成直接竞争关系
- 两者均采用 [[Agentic Loop]] 模式和 [[MCP协议]]
- GitHub Copilot 依托 Microsoft 生态，Claude Code 依托 Anthropic 的模型能力

### 技术影响

- Copilot 系列产品推动了 AI 编程助手的普及
- Copilot CLI 展示了终端 AI 助手的工程代理能力
- 对 [[MCP协议]] 的支持加速了协议的生态建设

Shell 编程使用 Bash 等 Shell 解释器编写脚本，通过组合系统命令、流程控制和文本处理工具实现系统管理自动化。核心内容包括重定向与管道、Bash 脚本语法（变量/条件/循环/函数/数组）、正则表达式，以及 grep、sed、awk 文本处理三剑客。Vim 编辑器是 Linux 下最强大的终端文本编辑工具。

## 定义

Shell 编程是指使用 Bash 等 Shell 解释器编写脚本，通过组合系统命令、流程控制和文本处理工具，实现系统管理自动化和数据处理。Bash（Bourne Again Shell）是 Linux 最广泛使用的 Shell，既是交互式命令行界面，也是脚本编程语言。

## 关键要点

### 重定向与管道

- **标准流**：stdin（0）、stdout（1）、stderr（2）
- **重定向**：`>` 覆盖写入、`>>` 追加、`2>` 错误重定向、`&>` 全部重定向、`<` 输入重定向
- **管道**（`|`）：将前一个命令的 stdout 作为下一个命令的 stdin，实现命令链式组合
- **Here Document**（`<<`）和 **Here String**（`<<<`）：多行输入和单行输入快捷方式

### Bash 脚本语法

- **变量**：用户变量（`VAR=value`，等号两侧无空格）、环境变量（`export`）、特殊变量（`$0/$1.../$#/$@/$?`）
- **条件判断**：`if/elif/else/fi`、`test` 命令与 `[ ]` / `[[ ]]`、`case/esac`
- **循环**：`for`（遍历/C 风格）、`while`、`until`
- **函数**：`function name() {}`，通过位置参数传参，`return` 返回退出码
- **数组**：索引数组（`arr=(a b c)`）和关联数组（`declare -A`）
- **退出状态**：`exit N`，0 表示成功，非 0 表示失败，`$?` 获取上一命令退出码

### 正则表达式

- **基础正则**（BRE）：`.` 任意字符、`*` 前一字符零次或多次、`^` 行首、`$` 行尾、`[...]` 字符集
- **扩展正则**（ERE）：`+` 一次或多次、`?` 零次或一次、`{m,n}` 重复次数、`|` 或、`(...)` 分组
- **常用场景**：日志过滤、配置文件修改、数据提取

### 文本处理三剑客

| 工具 | 定位 | 核心能力 |
|------|------|---------|
| **grep** | 文本搜索过滤 | 支持正则匹配、`-E` 扩展正则、`-v` 反选、`-r` 递归搜索 |
| **sed** | 流编辑器 | 按行处理文本、替换（`s/pattern/replace/g`）、删除（`d`）、插入（`i/a`）、`-i` 原地修改 |
| **awk** | 字段处理语言 | 按列处理数据、`NR` 行号、`NF` 字段数、`-F` 指定分隔符、支持条件和循环 |

典型组合用法：`grep` 粗筛 → `sed` 修改 → `awk` 提取统计

### Vim 编辑器

Vim 是 Linux 下最强大的终端文本编辑器，基于模式切换：

- **普通模式**：光标移动（hjkl/w/b/0/$）、删除（x/dd/D）、复制粘贴（yy/p/P）、撤销重做（u/Ctrl-r）
- **插入模式**：i/a/o/O 进入，Esc 退出
- **命令模式**：`:w` 保存、`:q` 退出、`:%s/old/new/g` 全局替换、`:set nu` 显示行号
- **可视模式**：v/V/Ctrl-v 字符/行/块选择
- **配置文件**：`~/.vimrc` 自定义行为（缩进、语法高亮、快捷键映射）
